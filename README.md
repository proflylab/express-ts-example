## Features

- Hot reload
- TypeScript
- Prettier
- ESLint

## Run Locally

Clone the project

```bash
  git clone https://github.com/proflylab/express-ts-example.git
```

Go to the project directory

```bash
  cd express-ts-example
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
